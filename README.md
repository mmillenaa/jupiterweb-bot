# jupiterweb bot usp 🤖🪐
Gerador de disponibilidade de dia e horário, créditos, quantidade de vagas e docentes em optativas eletivas.
Gera planilhas com todo o detalhamento durante o período de matrículas, em qualquer uma das interações.

## JupiterWebBot to extract course data
The script automates the extraction of course data from a web interface, including course credits, schedules, professors, vacancies, and enrollments, and generates a CSV file with this information.

### Como usar? 
1. Abra sua aba de matrícula dentro do período indicado no **calendário oficial** (não deixe para a última hora!).
2. Clique em **“Fazer matrícula”**.
3. Selecione a opção **“Optativas eletivas”**.
4. Pressione **Ctrl + Shift + I** para abrir as ferramentas de desenvolvedor.
5. Na lateral que aparecer, clique na aba **“Console”**.
6. Copie todo o código do arquivo **`index.js`** disponível neste repositório.  
   - Para acessar: abra o repositório, localize o documento `index.js`, clique nele e copie o conteúdo diretamente pelo GitHub.  
7. Cole o código copiado dentro da aba **“Console”**.
8. Aguarde alguns minutos até que o arquivo **CSV** seja gerado.  
   - ⚠️ **Atenção:** não recarregue a página até que o processo seja concluído.
