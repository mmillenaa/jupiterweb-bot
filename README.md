# jupiterweb-direito-usp 🤖🪐
Gerador de disponibilidade de dia e horário, créditos, quantidade de vagas e docentes em optativas eletivas.
Gera planilhas com todo o detalhamento durante o período de matrículas, em qualquer uma das interações.

## JupiterWebBot to extract course data
The script automates the extraction of course data from a web interface, including course credits, schedules, professors, vacancies, and enrollments, and generates a CSV file with this information.

### Como usar? 
1. Abra sua aba de matrícula no período indicado no calendário oficial (não deixe para última hora!).
2. Clique em "fazer matrícula"
3. Selecione "optativas eletivas"
4. Pressione Ctrl+Shift+I
5. Clique na aba "Console" da tela lateral que abrir para você
6. Cole lá todo o código do documento "index.js". Para acessar o código do bot, abra este reposiório e busque o documento "index.js". Clique nele, abra e copie aqui no Github mesmo. É isso que você cola na aba "Console".
7. Aguarde alguns bons minutos até que o csv seja gerado. Em hipótese alguma recarregue a página até que o trabalho esteja terminado.
